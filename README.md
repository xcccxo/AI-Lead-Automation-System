# AI-Lead-Automation-System
Automatización inteligente de leads desarrollado con n8n y Gemini AI.Captura leads en tiempo real mediante webhooks, los analiza con IA para asignar score y clasificación automática (Caliente, Tibio o Frío), almacena información en Google Sheets y envía alertas vía Slack con acceso directo a contacto por WhatsApp y agendamiento en Calendly.
